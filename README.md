# j1Lib_voice
simple text to voice via per character mp3

###Element


Example:

    button.style.display="none";
    j1Lib_voice(text,"v2","zh-hk",function(){
		button.style.display="block";
	});
	
   ![img](https://i.imgur.com/uvswssg.png)
    
###Parameter Reference
- text

- voice version

- language

- success callback
